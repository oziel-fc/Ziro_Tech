**Ziro Tech** é uma aplicação frontend de e-commerce desenvolvida para consumir dados de um serviço de web scraping externo, responsável por atuar como backend. A interface exibe produtos dinamicamente, permitindo navegação, busca e filtragem com foco em desempenho e experiência do usuário.

## Funcionalidades

- Layout **responsivo** para diferentes dispositivos.
- **Botão de ordenação** de produtos.
- Sistema de **pesquisa** de produtos.
- **Design intuitivo** e fácil navegação.
- **Menus dinâmicos** de categorias.
- Página de **produtos relacionados**.
- **Carrossel** de produtos e destaques.

## Sobre o projeto

Este projeto é uma aplicação de e-commerce desenvolvida com **Vue.js** e **TypeScript**, criada como parte do meu portfólio pessoal, com foco em **responsividade, desempenho** e **experiência do usuário**.
A aplicação consome dados de um serviço externo de **web scraping**, responsável por atuar como backend e fornecer os produtos dinamicamente. O objetivo é oferecer uma interface moderna, intuitiva e eficiente para navegação e descoberta de produtos.

## Aviso

Este projeto de e-commerce é uma aplicação pessoal, desenvolvida apenas para fins **educativos** e de **portfólio**, com o objetivo de demonstrar habilidades em **desenvolvimento frontend, design** e **consumo de dados**.

Os produtos e lojas exibidos são reais, porém a aplicação **não realiza vendas diretamente**. Ao selecionar um produto, o usuário será redirecionado para a **loja oficial** para concluir a compra.

Este projeto **não possui vínculo comercial** com as lojas ou marcas apresentadas, funcionando apenas como uma **vitrine demonstrativa**. O desenvolvedor não se responsabiliza por alterações de preços, disponibilidade dos produtos ou qualquer transação realizada em sites externos.
